# Tutor Landing Page

A simple GitHub Pages site for code tutoring services.

## How to use

1. Enable GitHub Pages in your repository settings
2. Choose the main branch as the source
3. Your site will be published at `https://yourusername.github.io/tutor-landing-page/`

## Customization

- Edit `index.md` to update the content
- Edit `_config.yml` to change site settings
- Add your avatar image to `assets/images/avatar.png`

## Local Development

To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`
